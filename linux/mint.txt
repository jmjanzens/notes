# Linux Mint Ubuntu Distro

---

### make text smaller
```bash
    ^+-
```

### restore text larger
```bash
    ^+Shift+\+
```

### about ttys 7 and 8 (Mint specific)
>   tty7 - upstart, launches Xorg, and occasionally hangs
>   tty8 - Cinnamon (GUI)

### difference between tty & pts
>   tty  - a regular terminal device
>   pts  - a psuedo terminal (xterm or ssh connection)

### cycle through ttys
```bash
    ^+Alt+[F1-6]
```

### quick switch between ttys
```bash
    Alt+[<-.->]
```
